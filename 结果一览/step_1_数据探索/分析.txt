Annual_Premium 原始分布图
高度右偏（长右尾）：
保费大部分集中在 20,000 到 50,000 左右；
有少数极端高值，甚至接近 400,000~500,000；
中心峰值非常尖，尾部很长，说明分布不均衡。
可能对模型产生影响（尤其是在选择树分裂点时）；
极端高值可能被忽略，或导致阈值选择不合理；
后续建议进行 对数变换（见图三）或分箱处理；
若存在极端离群值，可以设定阈值上限做裁剪（如 > 200k 截断）。

年龄分布图
年龄分布不规则但整体偏右：
年龄集中在 21-27 岁，峰值在 25；
有部分用户年龄高达 80+，但数量逐渐减少；
存在一些波动（可能和人口结构、业务需求有关）。

年龄段可能影响购买倾向（如年轻人或退休人群）；
可以考虑 年龄分段特征构造，如：
if age < 25: '青年'
elif age < 40: '中年'
elif age < 60: '中老年'
else: '老年'



Annual_Premium 对数变换后分布图
保费数据在对数变换后（log(Annual_Premium + 1)）：
明显更接近正态分布；
峰值更对称，尾部更短；
原来“高保费极端值”的影响被压缩。


非常适合用于建模，尤其是在使用：
线性模型（如 Logistic 回归、线性回归）；
模型解释与归一化可视化；
LightGBM 虽然不依赖线性关系，但对极端值仍敏感，因此：
建议使用对数变换后版本作为新的特征（不必替代原始值）；
例如新增 Annual_Premium_Log 作为新列，用于建模；




🖼 图 1：Driving_License 与 Annual_Premium 箱型图
🔍 观察结论：
有驾照（1）与无驾照（0）的保费分布差异不大；

两组中都存在大量异常值（高保费点）；

均值与中位数略有差异，但不明显。

🧠 启发：
Driving_License 本身可能不是决定保费高低的关键因素；

后续可以保留该特征，但单独作用有限，建议尝试与其他特征组合衍生新变量（例如：年轻 + 无驾照）；

🖼 图 2：Gender 与 Annual_Premium 箱型图
🔍 观察结论：
男女用户的保费分布几乎一致；

无论均值还是中位数，都无显著差异；

说明保险报价在性别上较为均衡。

🧠 启发：
性别特征区分度较低；

可以保留作为基础类别特征，但 不建议对其做 One-Hot 或复杂变换；

若要进一步利用，可尝试构造交互项（如性别 × 年龄段）；

🖼 图 3：Previously_Insured 与 Annual_Premium 箱型图
🔍 观察结论：
有无既往投保经验对保费分布几乎无影响；

两组的保费中位数、离群值、分布形状都相似。

🧠 启发：
Previously_Insured 更可能影响的是“是否购买”而不是“保费高低”；

说明它是一个重要的分类变量（对标签），但不是回归型特征；

对模型有信息贡献，建议保留。

🖼 图 4：Vehicle_Damage 与 Annual_Premium 箱型图
🔍 观察结论：
曾经出险（Yes）与未出险（No）用户的保费分布差异不大；

略微表现为：有事故历史的保费中位数稍高；

异常值分布相似。

🧠 启发：
可作为基础特征保留；

可与 Vehicle_Age 搭配组合，构建 车辆状况衍生变量（如：旧车 + 出过事故）；

🖼 图 5：Vehicle_Age 与 Annual_Premium 箱型图
🔍 观察结论：
车辆越老（>2 Years），保费略低；

中间年龄段（1-2 年）保费略高；

整体趋势是 车辆新旧对保费有一定影响，但不算太大。

🧠 启发：
可尝试将该变量转为顺序数值（如：<1 → 0，1-2 → 1，>2 → 2）；

也可用于衍生特征：如 “Vehicle_Age × Damage”；

🖼 图 6：Vintage 分布直方图
🔍 观察结论：
Vintage 分布非常均匀 → 每个天数的用户数量相对平均；

没有显著偏向。

🧠 启发：
Vintage 可以直接用于建模；

与其他变量组合使用。
